Descriptive statistics

  • Descriptive ≠ inferential, despite human tendency to jump to conclusions
  • RLS: tiny snapshot of the world
  • But if you don't do the descriptive part, inferential will suffer
  • The best value for description may not be the best for inference

Examples of descriptive statistics

Location: mean/median/mode/percentile rank

Spread: standard deviation/median absolute deviation/interquartile range

Shape: skewness/kurtosis

Dependence: Pearson/Spearman correlation coefficient

Recommended reading:
Unlearning descriptive statistics

  • Location/central tendency/average
  • Spread/variation/variability/dispersion
  • Shape: exotic, hard to interpret, better use an image
  • Dependence: danger zone! (association/causation) again, a scatterplot may do a better job

Describing our dataset with pandas

  • pandas is just a tool, but it's very convenient and powerful
  • Can be nonsensical: Key and SurveyID – but count is useful for missing values
  • Good sanity checks: Block has no missing values, but is between 0 (??) and 2
  • Latitude/longitude: Nice, but they often make sense together
  • Depth/Total are informative (more on that soon)
  • What about non-numerical columns? Worth running sanity checks on date range, missing values, value counts

Let's make the numbers more visual!

  • Similar effort to describe() with pandas
  • Note the different y-ranges!
  • Box: Q1 to Q3 with a line at Q2 (median)
  • The whiskers extend from the edges of box to show the range of the data
  • The position of the whiskers is set by default to 1.5 * IQR (IQR = Q3 - Q1) from the edges of the box
  • Outlier points are those past the end of the whiskers
  • Classic box plots give us the five number summary: min, max, Q1, median, Q3
